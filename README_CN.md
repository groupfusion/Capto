# Capto

一款轻量级、高性能的 Windows 截图工具，支持多种标注和编辑功能。

![.NET](https://img.shields.io/badge/.NET-9.0-purple.svg)
![Platform](https://img.shields.io/badge/platform-Windows-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## ✨ 特性

- 🚀 **轻量级高性能** - 基于 .NET 9.0 Windows Forms，内存占用低，启动快速
- 🎨 **7种绘图工具** - 画笔、矩形、圆形、箭头、模糊、文本、擦除
- 🖼️ **区域截图** - 拖拽选择截图区域，自由调整大小
- ⌨️ **快捷键支持** - 默认 Alt+S，可自定义快捷键
- 📋 **一键复制** - 复制到剪贴板，快速分享
- 💾 **PNG保存** - 自动生成文件名，保存到本地
- 🎯 **高DPI支持** - 完美支持高分辨率显示器
- 🎨 **SVG图标** - 使用 SkiaSharp 渲染，图标清晰美观
- 🔧 **可拖拽工具栏** - 工具栏位置灵活调整

## 📸 功能展示

### 截图功能
- 全屏遮罩，拖拽选择区域
- 支持拖拽调整区域大小
- 截图完成后显示工具栏

### 绘图工具

| 工具 | 功能 |
|------|------|
| ✏️ 画笔 | 自由绘制线条 |
| ⬜ 矩形 | 绘制矩形框 |
| ⭕ 圆形 | 绘制圆形/椭圆 |
| ➡️ 箭头 | 绘制箭头指示 |
| 🔲 模糊 | 马赛克/模糊效果 |
| 📝 文本 | 文字输入和标注 |
| 🧹 擦除 | 橡皮擦效果 |

### 工具属性
- **粗细选择**: 1px - 20px 多种画笔粗细
- **颜色选择**: 红、黑、蓝、绿、黄等多种预设颜色
- **箭头样式**: 可选择箭头样式

### 图像操作
- **保存**: PNG 格式，自动生成文件名
- **复制**: 一键复制到剪贴板
- **取消**: 点击关闭按钮仅关闭截图窗口

## 🛠️ 技术栈

- **框架**: .NET 9.0 Windows Forms
- **图形处理**: SkiaSharp (SVG图标渲染)
- **UI组件**: System.Windows.Forms
- **编程语言**: C#

## 📦 安装

### 前置要求
- Windows 10 或更高版本
- .NET 9.0 Runtime

### 安装步骤
1. 下载最新版本的 Capto.exe
2. 运行应用程序
3. 图标出现在系统托盘

## 📖 使用说明

### 快速开始
1. 运行应用程序，图标出现在系统托盘
2. 按 **Alt+S** 或点击托盘图标选择"截图"
3. 拖拽选择截图区域
4. 使用工具栏进行标注
5. 点击保存或复制完成操作

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| Alt+S | 截图 |
| Esc | 取消截图 |

### 系统托盘
- **左键双击**: 打开设置窗口
- **右键菜单**:
  - 截图: 开始截图
  - 设置: 打开设置窗口
  - 退出: 退出应用程序

### 设置
- 支持自定义截图快捷键
- 默认快捷键: Alt+S

## 🏗️ 项目结构

```
Capto/
├── Program.cs                 # 应用程序入口点
├── TrayIcon.cs               # 系统托盘图标管理
├── ScreenCaptureForm.cs      # 截图主窗口
├── ToolStripManager.cs       # 工具栏管理器
├── SettingsForm.cs           # 设置窗口
├── TransparentTextBox.cs    # 透明文本框组件
├── Base64Converter.cs        # Base64转换工具
├── DrawObjects/              # 绘图对象目录
│   ├── DrawObject.cs         # 绘图对象基类
│   ├── PenDrawObject.cs      # 画笔工具
│   ├── RectangleDrawObject.cs # 矩形工具
│   ├── CircleDrawObject.cs   # 圆形工具
│   ├── ArrowDrawObject.cs    # 箭头工具
│   ├── BlurDrawObject.cs     # 模糊/打码工具
│   ├── TextDrawObject.cs     # 文本工具
│   └── EraserDrawObject.cs   # 擦除工具
├── Utilities/
│   └── IconLoader.cs         # SVG图标加载器
└── resource/                 # SVG图标资源目录
```

## 🎯 项目特点

### 1. 轻量级高性能
- 基于 .NET 9.0 Windows Forms，内存占用少
- 单一可执行文件，启动快速
- 使用 Graphics 绘图，性能高效

### 2. 高DPI屏幕支持
- 应用程序启动时自动设置 DPI 感知
- 在高分辨率显示器上保持清晰

### 3. 完整绘图工具链
提供 7种专业绘图工具：画笔、矩形、圆形、箭头、模糊、文本、擦除

### 4. 现代化UI设计
- SVG 图标渲染 (SkiaSharp)，图标清晰美观
- 半透明工具栏
- 可拖拽工具栏位置
- 颜色和粗细一键切换

### 5. 低耦合架构
- 事件驱动设计: 工具栏与主窗口通过事件通信
- 面向对象: 绘图对象继承体系，易于扩展新工具

### 6. 系统集成度高
- 系统托盘常驻后台
- 支持 Alt+S 快捷键截图
- 支持复制到剪贴板

### 7. 自定义能力强
- 可自定义截图快捷键
- 工具栏可拖拽，适应不同场景
